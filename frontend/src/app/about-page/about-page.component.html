What is Blazebase?

Blazebase is a simple web application that allows users to find out where fires are currently burning and report nearby fires
The application allows for quick and autonomous user submission screening. Eventually, the application will provide fire forcasting using historical data
Blazebase makes participating in prevention simple and easy, just snap a picture of a fire and you are letting the rest of the world know.
